# Fix Dashboard Auth Logout V2 - Checklist

- [x] axios 拦截器的 401 处理检查 localStorage 中是否仍存在 token
- [x] 401 发生时显示明确的用户提示，而不是立即清除 token 并登出
- [x] 用户有选择是否重新登录的机会，而不是被强制登出
- [x] HomeView 中每个 API 调用有独立的错误处理
- [x] 单个 API 返回 401 时不触发全局登出
- [x] 用户能看到哪些数据加载失败，但保持登录状态
- [x] 401 错误发生时有详细的日志记录（请求 URL、响应状态、错误详情）
- [x] 用户登录并导航到其他页面后，返回仪表盘时不会被自动登出
