# Checklist

- [x] PPT 导出：`GET /api/v1/reports/{id}/export?fmt=pptx` 返回有效的 .pptx 文件，内容包含报告标题和各章节
- [x] 工作流自动入库：发起一次完整工作流分析后，数据库中 reports 表自动新增一条记录
- [x] 后端配置 API：`GET /api/v1/settings` 返回当前运行时配置，`PUT /api/v1/settings` 可更新配置
- [x] 运行时配置生效：修改 LLM provider/model 后，新发起的工作流使用新配置
- [x] 前端导出下拉菜单：工作流结果区域展示格式选择下拉菜单（Markdown/DOCX/PDF/PPTX），选择后触发下载
- [x] 智能报告页面可访问：左侧导航"智能报告"菜单项可点击并正确路由到 /reports
- [x] 报告列表展示：智能报告页面展示所有已生成报告，包含标题、时间、操作按钮
- [x] 报告搜索：输入关键词后列表过滤匹配的报告
- [x] 报告预览：点击预览按钮弹窗展示 Markdown 渲染内容
- [x] 报告标题编辑：内联修改标题后调用 PUT 接口更新，刷新列表反映变更
- [x] 报告删除：点击删除并确认后报告从列表消失且数据库已清除
- [x] 报告多格式导出：从智能报告页面选择报告导出，四种格式均可正确下载
- [x] 设置页加载配置：`SettingsView.vue` 打开后从后端加载当前配置并填充表单字段
- [x] 设置页保存生效：修改 Provider/Model/Temperature 等参数后保存，后端配置更新
